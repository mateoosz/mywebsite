What I've learned

Here are some examples of tags I've learning during this programme:

  • p tag - creates new paragraph

  • img tag - loads an image you chose

  • a tag - adds a link both local (relative) or out of your page (absolute)

  • table tag - creates a table

The list goes on further, of course. You can see a favicon next to my page URL. It was created by using link tag, so my page can be one of the kind and is associated with this picture. It's like a logo to my page.

The most exciting thing I learned so far is being able to embed into my website things like YouTube videos (which I did up higher using iframe tag) or even Google Maps!

Apart from HTML I learned a lot about CSS so far! In fact, my webpage is decorated using this language!
